> Get a [BananaPi Zero](http://www.banana-pi.org/m2z.html) (rasp pi clone with video hardware) and install the cast software.

Sway Cast is simply raspbian image with Xorg. It borrows feh  and syncthing  (for wallpapers management)  and polybar dotfiles from sway workstation. It supports [DLNA](https://github.com/hzeller/gmrender-resurrect) and [CastV2](https://github.com/thibauts/node-castv2).


We require you use  [banana pi m2 zero](http://www.banana-pi.org/m2z.html). The raspberry pi does not have the necessary specs.

## Requisites
### Hardware
- [ ] [banana pi m2 zero](http://www.banana-pi.org/m2z.html)
- [ ] microSD: At least 64G
- [ ] [Mini Hdmi to HDMI cable](https://www.amazon.com/gp/product/B0732QMFQ2) (the shorter the better)
- [ ] [dumb tv](https://geekydeck.com/best-dumb-tv/) (optional)

## Installation

1. Download the img from from the  [here](https://minio.sway-me.xyz/cast/cast.img).

2. Run (replacing `X` with the letter assigned to your microSD):
```
sudo dd bs=4M if=cast.img  of=/dev/sdX status=progress
```
3. Insert into BananaPi and boot up.

4. Test by booting up and casting from your local jellyfin instance.
